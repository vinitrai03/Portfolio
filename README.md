Feel free to check out the live version of this portfolio here: https://vinitrai03.netlify.app/



---

## 🏗️ Architecture & Features

This portfolio avoids heavy frameworks and instead relies on a clean, vanilla JavaScript architecture to simulate a snappy component-based frontend.

- **Component-Based Vanilla JS:** HTML sections (Hero, About, Projects, etc.) are dynamically fetched and injected into the DOM, making the codebase incredibly easy to maintain and edit.
- **Glassmorphism Design Theme:** A sleek, dark-mode-first aesthetic with dynamic neon cyan accents and frosted glass effects.
- **Scroll Animations:** Integrated with `AOS.js` for smooth, scroll-triggered fade-ups and zoom-ins.
- **Interactive Modals:** Project details open in beautifully designed, animated modals without ever leaving the page.
- **Clickable Achievements/Certifications:** Clean UI cards that link directly to official PDF certificates.

## 🚀 How to Run Locally

Since this project fetches HTML components locally via JavaScript, running it directly by double-clicking `index.html` might cause CORS errors in some browsers. To view the site perfectly:

### Prerequisites:
- Have [Visual Studio Code](https://code.visualstudio.com/) installed.

### Steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/vinitrai03/Portfolio.git
   ```
2. Open the folder in VS Code.
3. Install the **Live Server** extension in VS Code.
4. Open the `index.html` file.
5. Click **"Go Live"** in the bottom right corner of VS Code.
6. The portfolio will instantly open in your default browser at `http://127.0.0.1:5500`.
